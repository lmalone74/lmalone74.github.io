# Artifact Three - Grazioso Salvare Database
**Database Enhancement**

A full-stack dashboard application enhanced with secure database integration, modular architecture, and interactive data visualization to support efficient data management.
>
### [View Original Code (GitHub)](https://github.com/lmalone74/Artifact-Three-Grazioso-Salvare/tree/Original-Code) | [View Enhanced Code (GitHub)](https://github.com/lmalone74/Artifact-Three-Grazioso-Salvare/tree/Enhanced-Code)
>

For my third artifact, I selected the Grazioso Salvare Animal Shelter Dashboard from CS 340 (2025). It is a Python-based application that connects to a MongoDB database and allows users to view, filter, and analyze animal shelter data through an interactive table, chart, and map. The original version was created in a Jupyter Notebook as part of a database assignment.

I chose this artifact because it demonstrates my ability to work with databases, integrate backend data with a user interface, and build a data-driven application. It highlights my use of CRUD operations, data handling with pandas, and interactive visualization using Dash. The enhancement focused on improving structure and security by refactoring the application into separate modules, removing hardcoded credentials in favor of environment variables, adding a login system, and implementing input validation in the CRUD layer.

Through this process, I learned the importance of modular design and securing application data. A key challenge was restructuring the notebook into a multi-file application while maintaining functionality, which required careful testing. Feedback from earlier coursework influenced my decision to improve code organization and eliminate hardcoded values. These changes improved the reliability, maintainability, and security of the application.

This enhancement primarily supports Outcome 5, focusing on secure computing solutions, through the use of environment variables, access control, and data validation. It also supports other outcomes related to system design and data presentation.
